How to Use Jot It

Creating Notes

Tap the yellow + button to create a note. Type your content or use the Scan tab for handwriting.

Handwriting Recognition

Use the Scan tab to draw or upload handwritten notes. OCR converts your handwriting to text.

  • Write clearly for best results
  • Use dark ink on light background
  • Avoid crossed-out text

Drawing Tips

  • Select pen size (small / medium / large)
  • Choose black for best OCR accuracy
  • Use eraser to fix mistakes
  • Clear resets the canvas

Organizing Notes

Use category tabs — Personal, Work, Ideas — or add hashtags like #work in content.

Search

Tap the Search icon in the bottom nav to search by title or content.

Keyboard Shortcuts

  • Ctrl + K — Command palette
  • Ctrl + N — New note
  • Ctrl + F or / — Search
  • Ctrl + S — Save

About This App

Jot It is an intelligent note-taking app that combines a clean writing experience with AI-powered handwriting recognition. Draw or upload handwritten notes and let Azure Computer Vision convert them to text instantly.

Features

Rich text editor with formatting
Handwriting OCR via Azure Vision
AI-generated note summaries
6 note templates
Pin and organize notes
Export as PDF or CSV

Tech Stack
